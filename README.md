# Review Insight - Iniciação Científica 
Desenvolvimento de sistema para a obtenção e análise automatizada da experiência turística para o Museu Oscar Niemeyer e Hospital Erastinho a partir de plataformas de compartilhamento de opiniões

## Créditos aos projetos que auxiliaram no desenvolvimento do software
    https://github.com/xxmarce1oo/IC-Mineracao-de-Dados-Hospitalares 
    https://github.com/JoaoPedroMDP/review-scraping

## Como baixar 
1. Clone o repositório
2. Instalar dependencias `pip install -r requirements.txt`
